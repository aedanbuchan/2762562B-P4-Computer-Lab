# Demos Folder ReadME

## This folder contains demo notebooks for each set of functions developed within this project.
Users are encouraged to download the notebooks and run them to better understand the functions.
All notebooks but "pipeline.ipynb" run using simulated data generated by the "sim.py" functions, this allows for users to experiment without acess to the original data.

"pipeline.ipynb" uses the real azimuthal dataset used throughout the project and demonstrates the main findings of the project, while giving insight into the utility of the package on real data.
